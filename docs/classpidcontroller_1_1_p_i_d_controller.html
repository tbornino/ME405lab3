def pidcontroller.PIDController.run (   self)

Continuously runs the control algorithm.

Reads the position data from a sensor and then finds the error between the actual position and the desired setpoint value. Then we append the stored data list with a tuple of values.

Returns
The actuation value to fix steady state error.
